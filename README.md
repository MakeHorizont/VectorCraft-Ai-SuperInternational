## ЁЯЪА Key Features

### 1. The Genesis Mode (Creation)
Describe an object, define a visual style (e.g., "Cyberpunk", "Flat Design"), and attach reference images. Gemini 3 Pro will engineer a clean, semantic SVG code.

### 2. The Transformation Mode (Edit)
Upload an existing SVG file or paste the code. Provide instructions to modify it.
*   *Example:* "Make the lines thicker and animate the gears spinning."
*   The AI understands the structure and strictly modifies properties without breaking the file.

### 3. Dialectical Aufhebung (Quick Refine)
Located beneath the preview. If the result has a flaw (a "contradiction"), simply describe it. The system will preserve the good parts and fix the flaw, creating a new synthesis.

### 4. Logistics & Export
*   **Drag & Drop:** Drop images anywhere to use them as references.
*   **Smart Clipboard:** Paste images directly with `Ctrl+V`.
*   **Export:** Download as source `.svg`, rasterized `.png`, or a bundled `.zip`. All files are automatically named with a slug and timestamp version.

### 5. Tech Stack
*   **Audio/Animation:** Supports embedding CSS animations and SMIL. *Note: Script tags are sanitized for security, interactivity is handled via inline DOM APIs.*
*   **Search Grounding:** Optional toggle to allow the AI to search Google for up-to-date visual references.

## ЁЯЫая╕П Installation & Setup

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/your-username/vectorcraft-ai.git
    cd vectorcraft-ai
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Configure API Key**
    Ensure you have a valid API Key for Google Gemini.
    *   *Note: This project is configured to use `process.env.API_KEY`. In a standard Vite/React setup, create a `.env` file:*
    ```env
    REACT_APP_API_KEY=your_gemini_api_key_here
    ```
    *(Or configure your bundler to inject the key variable).*

4.  **Start the Development Server**
    ```bash
    npm start
    ```
