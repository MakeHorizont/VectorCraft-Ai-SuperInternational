## ЁЯЗ║ЁЯЗ╕ English Documentation

**VectorCraft AI** is a next-generation workstation for vector graphics production. We bridge the gap between prompt engineering and visual design, allowing you to generate, edit, and export SVGs with ease.

### тЬи Key Features

#### 1. The Genesis Mode (Creation)
Create SVGs from scratch. Describe an object, define a visual style (e.g., "Cyberpunk", "Flat Design"), and attach reference images via **Drag & Drop** or **Paste (Ctrl+V)**.

#### 2. The Transformation Mode (Edit)
Upload an existing SVG file or paste the code. Provide instructions to modify it (e.g., *"Make the lines thicker and animate the gears spinning"*). The AI understands the code structure and applies changes without breaking the file.

#### 3. Dialectical Aufhebung (Refinement)
Located beneath the preview. If the result has a flaw (a "contradiction"), simply describe it. The system will preserve the good parts and fix the flaw, creating a new synthesis.

#### 4. Logistics & Export
*   **Resolution Control:** Set custom dimensions or standard presets (1K, FHD).
*   **Export Formats:** Download as source `.svg`, rasterized `.png`, or a bundled `.zip`. All files are automatically versioned with timestamps.

### ЁЯЫая╕П Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/vectorcraft-ai.git

# 2. Install dependencies
npm install

# 3. Setup API Key (in .env)
REACT_APP_API_KEY=your_gemini_api_key

# 4. Start the factory
npm start
```
